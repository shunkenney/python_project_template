```bash
bash setup.sh
```

# Project structure
## Flat layout (for small project)
```bash
my_proj/
    ├─ my_proj/
    │   ├─ __init__.py
    │   ├─ __main__.py
    │   ├─ cli.py
    │   ├─ submodules/
    │   :
    │
    ├─ config/
    ├─ scripts/
    ├─ .gitignore
    ├─ .gitmodules
    ├─ README.md
    └─ pyproject.toml
```
## Src layout (for big project)
```bash
my_proj/
    ├─ src/
    │   └─ my_proj/
    │       ├─ __init__.py
    │       ├─ __main__.py
    :       :
```
